# LLM_RAG_Workflows_Repo
A collection of AI automation workflows using LLMs, RAG techniques, and multi-step agent-based processes built on n8n. These workflows aim to reduce human effort and increase efficiency.

AI Automation & Agentic Workflow Portfolio

Overview

This repository contains a collection of AI-powered automation workflows built using n8n, Google Gemini, OpenAI APIs, RAG architectures, and various no-code/low-code tools.

The projects focus on reducing manual effort, improving operational efficiency, and enabling intelligent decision-making through AI agents, workflow automation, and document intelligence systems.

Featured Projects

1. Cold Outreach Email Automation

Problem:
Sending bulk cold emails and outreach at scale is tedious and error-prone.

Solution:
This workflow automates personalized bulk email outreach. It uses a list of recipients from a data source (like Google Sheets), dynamically crafts cold emails, and schedules or sends them in bulk, ensuring personalization and efficiency.

Tech Stack:
n8n, Gmail API, Google Sheets

⸻

2. Property Analysis Agent

Problem:
Analyzing property investment ROI and generating reports often involves manual calculations.

Solution:
This agent, triggered via a POST request, automates property ROI calculations. It processes inputs, uses an LLM for report generation, and provides a detailed property analysis output.

Tech Stack:
n8n, LLM (e.g., OpenAI/Gemini), Postman (for trigger), Webhooks

⸻

3. Weekly Business Report Agent

Problem:
Businesses in sectors like restaurants or gyms need regular performance reports but waste time doing it manually.

Solution:
This workflow pulls data (e.g., sales, customer counts) from Google Sheets, analyzes trends, and generates a weekly business performance report. The report is sent via email to stakeholders, providing actionable insights.

Tech Stack:
n8n, Google Sheets, Gmail API, Gemini API
